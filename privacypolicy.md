## Personal Information

AppVaultX **does not collect, store, or share any personal information**.  
We do not require user accounts, and we do not process or transmit any identifiable data to external servers.

If any personal data is inadvertently received (e.g., through user communication), it will **not** be shared, sold, or used for any purpose beyond addressing the specific inquiry.

## Permissions

AppVaultX requests the following permissions solely to enable core functionality:

1. **`QUERY_ALL_PACKAGES`** – Used to list and manage installed applications on the device. This is necessary for the app’s core functionality and is not used to track or profile users.
2. **`WRITE_EXTERNAL_STORAGE`** – Allows AppVaultX to save exported APK files to local device storage (applies to Android versions below Android 10, where scoped storage is not enforced).
3. **`moe.shizuku.manager.permission.API_V23`** – Enables integration with the [Shizuku](https://github.com/RikkaApps/Shizuku) service for executing privileged operations safely without requiring root access.

AppVaultX **never uploads or transmits** the data accessed through these permissions.

## Data Security

AppVaultX performs all operations **locally on the user’s device**.  
No data is sent to external servers, ensuring that user activity and app data remain private and under the user’s control.
